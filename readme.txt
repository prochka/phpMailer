Как пользоваться скриптом

1. Скачать библиотеку PHPMailer здесь: https://github.com/Synchro/PHPMailer
2. Подключаем PHPMailerAutoload.php
3. И далее по примеру здесь: http://phpmailer.worxware.com/index.php?pg=examplebmail
4. SMTP для каждого почтовика свой. В поиск: smtp mail.ru или smtp google.com
5. Все методы и свойства объекта $mail здесь: http://phpmailer.worxware.com/
